Idempotence

The integration service ensures idempotence.

Examples

Handle already processed successful transaction

If a GET /handle/{id} is requested for a Payment Object whose most recent transaction status is Success then the response is HTTP/1.1 204 No Content and the body is empty.

Handle already processed failed transaction

If a GET /handle/{id} is requested for a Payment Object whose most recent transaction status is Failure then the response is HTTP/1.1 200 OK and the body is empty.
