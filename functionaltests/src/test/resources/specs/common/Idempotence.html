Idempotence

The integration service ensures idempotence.

Examples

Common Setup

https://api.pay1.de/post-gateway/ is used as PAYONE API URL.

Handle already processed successful transaction

Given a payment p1 with transaction status Success .

When I send GET /commercetools/handle/payments/p1 .

Then

  • I should receive HTTP/1.1 200 OK
  • with response containing {"statusCode": 200, "newPaymentVersion": 1} and
  • the number of requests sent to the PAYONE API URL is 0.

Handle already processed failed transaction

Given a payment p2 with transaction status Failure .

When I send GET /commercetools/handle/payments/p2 .

Then

  • I should receive HTTP/1.1 200 OK
  • with response containing {"statusCode": 200, "newPaymentVersion": 1} and
  • the number of requests sent to the PAYONE API URL is 0.
